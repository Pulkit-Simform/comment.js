 $B.ui.dialog.create(options)✰➭

Creates a ui dialog with given options

Arguments

options
object
object of name/value pairs
  1. {
    1. elstringselector for the element to act as a dialog (optional)
    2. idstringunique id for the dialog
    3. contentstringstring content for de dialog (could be the inner html of a div $('#myDiv').html())
    4. afterLoadcallbackcallback function to execute after loading dialog content via ajax
    5. buttonobject, stringjQuery object or selector for the button that triggers the dialog
    6. urlstring, functionurl string or function that returns a url string
    7. urlParamsobject, functionobject or function that returns an object with params for the url
    8. tagNamestringthe tagName for creating the dialog if no element was provided (div by default)
    9. dialogOptionsobjectAny jQuery ui option (see jquery ui documentation)
    10. onCreatefunctioncallback that fires only once when the dialog is created
  2. }

Returns: object jQuery object of the dialog content

Usage 

$B.ui.dialog.create({
  id: 'MyDialog',
  el: '#DialogElement',
  button: '#button',
  dialogOptions: {
    draggable: true,
    width: 640
  }
 });
